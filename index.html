Below are functions that you can run to access the API data.

"GET" /users = returns all users' json data
"GET" /users/:id = returns json data for user of id given
"GET" /tasks = returns all tasks' json data
"GET" /tasks/:id = returns json data for task of id given
"POST" /users = create a new user
"POST" /tasks = create a new task
"PATCH" /in-progress = changes status to "In Progress"
"PATCH" /testing = changes status to "Testing"
"PATCH" /completed = changes status to "Completed"
"PATCH" /password = updates password of user
"PATCH" /promote = promotes user
"PATCH" /demote = demotes user
